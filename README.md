**Purpose:** This website demonstrates how Q-Learning and Double Q-Learning guide an agent to park safely while avoiding obstacles. Users can compare both algorithms’ routes, steps, turns, and performance in different parking environments.

<img width="1748" height="891" alt="image" src="https://github.com/user-attachments/assets/3f2ae3db-aa9f-4ce3-b3a6-ff25329af680" />

**How to Use:** Select a difficulty level, set the map seed, maximum steps, and speed, then click **Start** to run the simulation. Use **Pause** to observe the agent’s decisions and **Reset** to restart the simulation. Compare the two maps and live metrics to see which algorithm performs better.
